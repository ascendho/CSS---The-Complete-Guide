Question 2

What does the following syntax do?

a.active { 
    color: white;
}

1
2
3

Correct Answer

1

It will set a white text color on all <a href="..." class="active">  elements.

Explanation

This is correct, the shown syntax requires that the class active is added directly to the element.

Question 3

What does the following syntax do?

.active,
a:hover {
    color: white;
}

1
2

Correct Answer

2

It'll set a white text color ONLY to elements like this:

<a href="..." class="active">Click me!</a> 

but also to

<a href="...">Click me!</a>  IF the user hovers over it.

Explanation

This is correct. The shown syntax groups two CSS rules - the first selector targets all elements which also have the "active" class, the second selector targets all elements over which the user hovers.
